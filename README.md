# 🚀 Build Five Programs (Bash)

This repository contains 5 different Bash scripts developed during the freeCodeCamp course. 
Each script demonstrates different programming concepts in a Linux environment.

## 📁 Included Programs
1.  **questionnaire.sh**: Handles user input and variables.
2.  **countdown.sh**: Uses `while` loops for a timer function.
3.  **bingo.sh**: Uses random number generation and `if/else` logic.
4.  **fortune.sh**: A fun interactive script with conditional outputs.
5.  **five.sh**: The master script that executes all the above programs.

## 🛠 Skills Demonstrated
* Control Flow (If/Else, While loops, Case statements)
* Input/Output Handling
* Script Automation & Modularization
